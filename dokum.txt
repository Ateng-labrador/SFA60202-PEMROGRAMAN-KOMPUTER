main -> visualisasi
Pendulum -> model 1 bandul
equation -> ODE + turuna
double_pendulum -> gabungan dua bandul,panggilana solver


* Pendulum

Pendulum memiliki class(L,m)

calcu_path menerima arrat theta dan dtheta untuk menghitung 
posisi ujung bandul.Menyimpan arrays,x,y, dan dataframe dan 
menyediakan getter max


* equation

solve_ode membungkus scipy.intgegrate.odeint, meneruskan args = 
(L1,L2,m1,m2,g).

* double_pendulum
Kelas double_pendulum membuat dua objek Pendulum
Menyimpan waktu t (kelas), mengubah y0 ke radian

_calculate_sytem:
    memanggil solve_ode -> menghasilkan y array ukuran (len(t),4).
    memanggil Pendulum1.calcu_path(theta = y[:,0], dtheta = y[:,1])
    memanggil Pendulum2.calcu_path(theta = y[:,2], dtheta = y[:,3], x0=pendulum1.x, y0=pendulum1.y)
    sehingga posisi bandul ke-2 offset oleh posisi bandul ke-1


    menyimpan dataframe dan beberapa accessor untuk frame/animasinya


* main

create_double_pendula() membuat satua atau banyak double_pendulum

create_axes() membuat subplot untuk setiap Pendulum,menggambar line awal.

FuncAnimation memanggil animate(i) setiap frame:
    ** animate membaca frame_x, frame_y dari each double_pendulum untuk index i
    ** meng-update line.set_data(frame_x, frame_y) dan time_text
Hasil: animasi gerak chain dari origin -> Pendulum1 -> Pendulum2

Aliran data / ukuran

y: array (nt 4),dengan kolom [theta1,z1,theta2,z2]
pendulum.x,Pendulum.y: array (nt) tiap bandul
get_frame_coordinates untuk plotting: (0, x1[i], x2[i]) dan (0, y1[i], y2[i]).


def __init__ adalah parameter yang awal pada sebuah klass
